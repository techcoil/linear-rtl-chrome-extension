# linear.app-rtl

Minimal Chrome extension that injects RTL CSS and a small helper script on `https://linear.app/*`.

## Install (local)
1. Open `chrome://extensions`
2. Enable "Developer mode"
3. Click "Load unpacked" and select this folder

## Files
- `manifest.json` - Extension manifest (MV3)
- `content.js` - Sets `dir`/`lang` attributes
- `content.css` - RTL direction and input alignment
