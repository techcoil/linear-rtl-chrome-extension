# Privacy Policy  
**Linear RTL Chrome Extension**

_Last updated: 2026-02-03_

## Overview
Linear RTL Chrome Extension applies right-to-left (RTL) styling adjustments to the
Linear web application (`linear.app`). It is designed with privacy as a core
principle.

## Data Collection
The extension **does not collect, store, transmit, or process any data**.

- No user data
- No personal information
- No analytics or tracking
- No cookies
- No network requests

## Permissions and Functionality
The extension operates only on `linear.app` and injects a small amount of static
CSS to adjust layout direction.

- No page content is read
- No JavaScript processes user data
- No background scripts

## Third-Party Services
No third-party services, libraries, or APIs are used.

## Data Storage
The extension does not use cookies, local storage, sync storage, IndexedDB, or
any other storage mechanism.
