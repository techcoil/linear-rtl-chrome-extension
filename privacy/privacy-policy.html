Privacy Policy

Linear RTL Chrome Extension
Last updated: 2026-02-03

Overview

Linear RTL Chrome Extension is a lightweight browser extension that applies right-to-left (RTL) styling adjustments to the Linear web application (linear.app). The extension is designed with privacy as a core principle.

Data Collection

Linear RTL Chrome Extension does not collect, store, transmit, or process any personal data.

  • No user data is collected
  • No personally identifiable information (PII) is accessed
  • No analytics or tracking is performed
  • No cookies are read, set, or stored
  • No data is sent to external servers

Permissions and Functionality

The extension runs only on the linear.app domain. Its sole functionality is to inject a small amount of static CSS (approximately 10 lines) to adjust layout and text direction.

  • No page content is read
  • No JavaScript logic processes user data
  • No background scripts or network requests are used

Third-Party Services

This extension does not use third-party services, libraries, trackers, or APIs.

Data Storage

The extension does not use any form of storage, including:

  • Cookies
  • Local or sync storage
  • IndexedDB
  • Session storage

Changes

This policy will be updated if the extension’s functionality changes in a way that affects privacy.

Contact

For questions regarding this policy, please contact the developer via the Chrome Web Store listing.
